## <a name="edit-or-copy-powershell-scripts"></a><a name="bkmk_psedit"></a> Edición o copia de scripts de PowerShell
<!--3705507-->

Ahora puede **editar** o **copiar** un script de PowerShell existente que se usa con la característica [Ejecutar scripts](../../../../../apps/deploy-use/create-deploy-scripts.md). En lugar de volver a crear un script que necesita cambiar, ahora puede editarlo directamente. Ambas acciones utilizan la misma experiencia de asistente que cuando se crea un nuevo script. Cuando edita o copia un script, Configuration Manager no mantiene el estado de aprobación. 

> [!Tip]  
> No edite un script que se esté ejecutando activamente en los clientes, porque no terminarán de ejecutar el script original y es posible que no obtenga los resultados deseados de estos clientes.
