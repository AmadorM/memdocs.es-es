# <a name="capabilities-in-configuration-manager-technical-preview-version-18102"></a>Funcionalidades de la versión 1810.2 de Configuration Manager Technical Preview 

*Se aplica a: Configuration Manager (rama Technical Preview)*

En este artículo se presentan las características disponibles en la versión 1810.2 de Configuration Manager Technical Preview. Instale esta versión para actualizar y agregar características nuevas a su sitio de Technical Preview.
